Overview

Manthan is a deterministic decision infrastructure. Identical input produces identical output. Always.

Pipeline

1. Validation     → INVALID
2. Completeness   → INCOMPLETE
3. Evaluation     → ALLOW / BLOCK / ESCALATE
4. Explanation    → TRACE

Architecture

Definition → Schema + Rules
Signal     → External deterministic inputs
Core       → Pure evaluation
Token      → Signed proof
Gateway    → Execution control

Schema

{
  "required": ["amount", "role"]
}

Rules

IF amount <= 1000 AND role == admin
THEN ALLOW

Decision Token

{
  "outcome": "ALLOW",
  "signature": "..."
}

Constraints

- No hidden defaults
- No runtime mutation
- No AI in evaluation

Manthan enforces deterministic decision systems — where outcomes are reproducible, explainable, and auditable.
